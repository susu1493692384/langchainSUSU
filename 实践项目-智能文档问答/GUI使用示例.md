# 🖥️ 智能文档问答系统GUI使用示例

本示例将指导您如何使用智能文档问答系统的GUI版本。

## 📁 文件结构

```
langchain/
├── document_qa_gui.py      # 主GUI应用程序
├── 文档问答系统.py         # 原始问答系统
├── install_requirements.py # 依赖安装脚本
├── start_gui.py            # GUI启动器
├── GUI_README.md           # 详细使用文档
├── GUI使用示例.md          # 本文件
└── local_documents.txt     # 示例文档文件
```

## 🚀 启动步骤

### 方法1: 使用启动器（推荐）
```bash
python start_gui.py
```

### 方法2: 直接启动
```bash
python document_qa_gui.py
```

## 🎯 使用示例

### 示例1: 基本问答

1. **启动应用**
   - 运行 `python start_gui.py`
   - 等待系统初始化完成

2. **查看系统状态**
   - 右侧"系统监控"卡片显示：
     - 记忆节点数量
     - 当前活跃话题
     - 实体关系图谱
     - 性能指标

3. **进行问答**
   - 在底部输入框输入：`什么是人工智能？`
   - 点击"🚀 发送问题"或按`Ctrl+Enter`
   - 查看AI的回答和来源信息

4. **继续对话**
   - 输入：`它有哪些应用领域？`
   - 系统会自动理解"它"指代人工智能
   - 基于文档内容提供相关回答

### 示例2: 文档管理

1. **查看当前文档**
   - 左侧"文档管理"卡片显示已加载的文档
   - 点击"👁️"按钮预览文档内容

2. **上传新文档**
   - 点击"📤 上传文档"按钮
   - 选择.txt格式的文档文件
   - 系统自动处理和索引文档

3. **文档设置**
   - 调整"分块大小"参数（推荐300-500）
   - 系统会根据设置重新分割文档

### 示例3: 高级功能

1. **主题切换**
   - 点击顶部"🌙 暗色主题"按钮
   - 界面切换为暗色模式
   - 再次点击切换回亮色主题

2. **系统设置**
   - 点击顶部"⚙️ 设置"按钮
   - 调整：
     - 模型温度参数
     - 记忆系统配置
     - 文档检索设置

3. **导出对话**
   - 点击对话区域"💾 导出"按钮
   - 选择保存位置
   - 对话历史保存为文本文件

## 🔧 系统监控指标说明

### 记忆系统统计
- **记忆节点数量**: 当前系统存储的记忆单元数
- **最大节点数**: 系统允许的最大记忆节点数
- **重要性阈值**: 记忆节点被保留的最低重要性分数

### 当前话题权重
- 显示当前对话中活跃的话题
- 数字越大表示话题越重要
- 随着对话进行，权重会动态变化

### 实体图谱
- **实体总数**: 系统识别的实体数量
- **实体关系**: 显示部分实体与话题的关联关系

### 性能指标
- **文档数量**: 已加载和索引的文档数
- **文档分块**: 文档被分割的块数
- **对话轮次**: 当前的对话轮次统计
- **话题历史**: 系统追踪的话题变化历史

## ⚠️ 常见问题解决

### 问题1: 启动失败
**症状**: 提示"模块未找到"
**解决**:
```bash
python install_requirements.py
```

### 问题2: API错误
**症状**: 提示"API密钥错误"
**解决**:
1. 检查`.env`文件是否存在
2. 确保`GLM_API_KEY`和`GLM_BASE_URL`设置正确
3. 验证API密钥有效性

### 问题3: 回答不准确
**症状**: AI回答不符合预期
**解决**:
1. 确保文档内容相关
2. 提供更具体的问题
3. 检查文档质量
4. 调整检索参数

### 问题4: 界面显示异常
**症状**: 界面布局错乱或显示不全
**解决**:
1. 尝试切换主题
2. 调整窗口大小
3. 重启应用

## 📊 性能优化建议

### 1. 文档优化
- 使用清晰、结构化的文档
- 避免过长的单一段落
- 确保文档编码为UTF-8

### 2. 参数调优
- 分块大小: 300-500字符
- 检索数量: 3-5个文档
- 模型温度: 0.3-0.7

### 3. 系统维护
- 定期清理不需要的记忆节点
- 及时更新文档内容
- 监控系统性能指标
